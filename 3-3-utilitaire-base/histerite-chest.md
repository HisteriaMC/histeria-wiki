# Coffre en Histerite

## Description

Le coffre en Histerite est utile pour contenir beaucoup d'objets en un seul coffre. En effet, le coffre en Histerite a 81 emplacements, soit l'équivalent d'un double coffre en plus d'un coffre simple, malgré le fait qu'il ne compte que pour un seul coffre sur le [chercheur de coffre](https://histeria.fr/wiki/3-1-utilitaire-principal/chest-finder).
